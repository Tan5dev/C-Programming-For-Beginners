# Unit Converter 📏

## Description

A menu-driven Unit Converter built in C.

This project converts different units such as distance, temperature, and weight.

---

## Features

✅ Kilometers to Meters

✅ Meters to Centimeters

✅ Celsius to Fahrenheit

✅ Kilograms to Grams

✅ Menu Driven Interface

---

## Concepts Used

- Variables
- Input and Output
- Arithmetic Operators
- Loops
- Switch Statement

---

## Menu

```text
1. Kilometers to Meters
2. Meters to Centimeters
3. Celsius to Fahrenheit
4. Kilograms to Grams
5. Exit
```

---

## Example

```text
Enter Kilometers: 5

5.00 KM = 5000.00 Meters
```
