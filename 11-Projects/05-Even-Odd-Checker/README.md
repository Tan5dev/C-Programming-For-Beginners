# Even Odd Checker 🔢

## Description

A beginner-friendly C program that checks whether a number is even or odd.

---

## Features

✅ User Input

✅ Conditional Statements

✅ Modulus Operator (%)

✅ Instant Result

---

## Concepts Used

- Variables
- Input and Output
- Operators
- Conditional Statements

---

## How It Works

A number is:

- Even if it is divisible by 2
- Odd if it is not divisible by 2

The program uses:

```c
number % 2
```

to determine the result.

---

## Example 1

### Input

```text
Enter a number: 10
```

### Output

```text
10 is an Even Number.
```

---

## Example 2

### Input

```text
Enter a number: 15
```

### Output

```text
15 is an Odd Number.
```
