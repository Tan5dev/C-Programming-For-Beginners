# ATM Simulator 🏦

## Description

A menu-driven ATM Simulator built in C.

Users can check balance, deposit money, withdraw money, and exit the system.

---

## Features

✅ Check Balance

✅ Deposit Money

✅ Withdraw Money

✅ Insufficient Balance Protection

✅ Menu Driven Interface

✅ Realistic ATM Workflow

---

## Concepts Used

- Variables
- Input and Output
- Conditional Statements
- Loops
- Switch Statement
- Menu Driven Programming

---

## Menu

```text
===== ATM SIMULATOR =====

1. Check Balance
2. Deposit Money
3. Withdraw Money
4. Exit
```

---

## Example

```text
Current Balance = ₹10000

Deposit Amount = ₹2000

New Balance = ₹12000
```

---

## Withdrawal Example

```text
Current Balance = ₹10000

Remaining Balance = ₹8500
```
