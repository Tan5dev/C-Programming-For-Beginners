# Grade Calculator 🎓

## Description

A C program that calculates a student's grade based on marks entered by the user.

---

## Features

✅ User Input

✅ Grade Calculation

✅ Input Validation

✅ if-else-if Ladder

---

## Concepts Used

- Variables
- Input and Output
- Conditional Statements
- Logical Operators

---

## Grading System

| Marks | Grade |
|---------|---------|
| 90 - 100 | A+ |
| 80 - 89 | A |
| 70 - 79 | B |
| 60 - 69 | C |
| 50 - 59 | D |
| Below 50 | F |

---

## Example

### Input

```text
Enter Marks (0-100): 85
```

### Output

```text
Grade: A
```

---

## Example 2

### Input

```text
Enter Marks (0-100): 42
```

### Output

```text
Grade: F
```
