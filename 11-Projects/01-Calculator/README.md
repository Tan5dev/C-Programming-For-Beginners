# Simple Calculator 🧮

## Description

A beginner-friendly calculator built using C.

This project performs:

- Addition
- Subtraction
- Multiplication
- Division

---

## Concepts Used

- Variables
- Data Types
- Input and Output
- Operators
- Conditional Statements

---

## Example

Input:

```text
Enter first number: 10
Enter operator (+, -, *, /): *
Enter second number: 5
```

Output:

```text
Result = 50.00
```

## Features

✅ Addition

✅ Subtraction

✅ Multiplication
