# Simple Interest Calculator 💰

## Description

A beginner-friendly C program that calculates Simple Interest based on Principal Amount, Rate of Interest, and Time.

---

## Features

✅ User Input

✅ Mathematical Calculation

✅ Formatted Output

✅ Real-World Financial Formula

---

## Concepts Used

- Variables
- Data Types
- Input and Output
- Arithmetic Operators

---

## Formula Used

```text
Simple Interest = (Principal × Rate × Time) / 100
```

---

## Example

### Input

```text
Enter Principal Amount: 10000
Enter Rate of Interest: 5
Enter Time (Years): 2
```

### Output

```text
Simple Interest = 1000.00
```

---

## How It Works

1. User enters principal amount.
2. User enters interest rate.
3. User enters time period.
4. Program calculates simple interest.
5. Result is displayed.

---

## Learning Outcomes

This project teaches:

- Using formulas in programs
- Working with floating-point numbers
- Displaying formatted results
