# Contact Book V3 📞

## Description

A simple Contact Book application built in C.

Users can add contacts and view saved contacts using a menu-driven interface.

---

## Features

✅ Add Contact

✅ Store Name

✅ Store Phone Number

✅ View All Contacts

✅ Menu Driven Interface

---

## Concepts Used

- Arrays
- Strings
- Loops
- Conditional Statements
- Menu Driven Programming
- String Functions

---

## Menu

```text
1. Add Contact
2. View Contacts
3. Exit
```

---

## Example

```text
Enter Name: Dinesh
Enter Phone Number: 9876543210

Contact Added Successfully!
```

---

## View Contacts

```text
===== CONTACT LIST =====

1. Dinesh - 9876543210
2. Rahul - 9988776655
```
