# Multiplication Table Generator ✖️

## Description

A simple C program that generates the multiplication table of any number entered by the user.

---

## Features

✅ User Input

✅ Loop-Based Table Generation

✅ Clean Output

✅ Beginner Friendly

---

## Concepts Used

- Variables
- Input and Output
- Loops
- Arithmetic Operators

---

## Example

### Input

```text
Enter a number: 7
```

### Output

```text
7 x 1 = 7
7 x 2 = 14
7 x 3 = 21
7 x 4 = 28
7 x 5 = 35
7 x 6 = 42
7 x 7 = 49
7 x 8 = 56
7 x 9 = 63
7 x 10 = 70
```

---

## How It Works

1. User enters a number.
2. Program runs a loop from 1 to 10.
3. Each iteration prints the multiplication result.

---
