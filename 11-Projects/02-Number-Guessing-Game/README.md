# Number Guessing Game 🎮

## Description

A simple guessing game built in C.

The player must guess the correct number.

The program provides hints until the correct answer is found.

---

## Features

✅ User Input

✅ Conditional Statements

✅ Loops

✅ Hint System

✅ Win Detection

---

## Concepts Used

- Variables
- Input/Output
- Conditional Statements
- Loops

---

## How It Works

1. Program stores a secret number.
2. User enters a guess.
3. Program checks the guess.
4. If incorrect, a hint is displayed.
5. Game continues until the correct number is guessed.

---

## Example

```text
=== Number Guessing Game ===

Guess a number between 1 and 10: 5
Too Low!

Guess a number between 1 and 10: 9
Too High!

Guess a number between 1 and 10: 7

Congratulations! You guessed correctly.
```
