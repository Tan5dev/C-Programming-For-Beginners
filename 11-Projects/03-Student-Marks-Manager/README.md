# Student Marks Manager 📚

## Description

Student Marks Manager is a beginner-friendly C project that allows users to enter marks for 5 students and calculates useful statistics.

---

## Features

✅ Store marks using arrays

✅ Calculate total marks

✅ Calculate average marks

✅ Find highest marks

✅ Find lowest marks

✅ Display complete report

---

## Concepts Used

- Variables
- Arrays
- Loops
- Conditional Statements
- Input and Output

---

## Example

### Input

```text
Enter marks for student 1: 85
Enter marks for student 2: 90
Enter marks for student 3: 78
Enter marks for student 4: 92
Enter marks for student 5: 88
```

### Output

```text
Total Marks = 433
Average Marks = 86.60
Highest Marks = 92
Lowest Marks = 78
```

---

## How It Works

1. User enters marks for 5 students.
2. Program stores marks in an array.
3. Program calculates total marks.
4. Program calculates average marks.
5. Program finds highest and lowest marks.
6. Results are displayed.

---
