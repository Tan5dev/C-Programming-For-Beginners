# Simple Calculator V2 🧮

## Description

Simple Calculator V2 is a beginner-friendly C project that performs basic arithmetic operations using functions and a switch statement.

This project demonstrates how to organize code properly and avoid repetition by using functions.

---

## Features

✅ Addition

✅ Subtraction

✅ Multiplication

✅ Division

✅ Division by Zero Handling

✅ Function-Based Design

✅ Switch Statement

## Example Run

```text
=== Calculator V2 ===

```

---

## How It Works

1. User enters first number.
2. User enters operator.
3. User enters second number.
4. Program calls the appropriate function.
5. Result is displayed.
