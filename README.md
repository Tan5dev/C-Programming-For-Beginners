# 🚀 C Programming For Beginners

> A structured collection of C programs, exercises, and mini-projects designed to help beginners learn C Programming from scratch.

Whether you're a student, beginner programmer, or preparing for technical interviews, this repository provides a practical roadmap to mastering the fundamentals of C Programming through hands-on coding and real-world mini projects.

## ⚙️ Getting Started

### 1️⃣ Fork the Repository

Click the **Fork** button at the top-right corner of this repository.

### 2️⃣ Clone the Repository

```bash
git clone https://github.com/Tan5dev/C-Programming-For-Beginners.git
```

### 3️⃣ Open the Project

Navigate to any program file and compile it using GCC.

### Compile

```bash
gcc filename.c -o output
```

### Run (Linux/macOS)

```bash
./output
```

### Run (Windows)

```bash
output.exe
```
